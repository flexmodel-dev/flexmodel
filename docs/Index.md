# Flexmodel 文档

欢迎来到 Flexmodel 文档中心！Flexmodel 是面向下一代应用的统一数据访问层。

## 导航

- [文档](/docs/tutorial/intro) — 使用教程和功能介绍
- [API](/docs/api) — API 文档和技术参考
- [部署](/docs/tutorial/deployment) — 运维部署指南

## 核心功能

| 功能 | 说明 |
|------|------|
| 数据建模 | 运行时动态建模，自动生成 API |
| 流程编排 | BPMN 风格可视化流程设计 |
| 任务调度 | Cron/延时/事件驱动调度 |
| 对象存储 | S3/本地统一存储抽象 |
| 身份认证 | RBAC + OIDC/LDAP/JWT |
| AI 工具集成 | MCP 协议开放核心工具 |
| 云函数 | Deno Functions Runtime 隔离执行（实验性） |
